# YGCVideoToolbox 

A series of video tools base on AVFoundation framework.



### Features
- [x] Resize video
- [x] Crop video by timerange
- [x] Reverse a video
- [x] Slow motion video
- [x] Repeat a video segment
- [x] Add CoreImage filter to a video file
- [x] Add image on video
- [x] Add gif on video
- [x] Add text on video
- [x] Pure Swift 4.

## Installing

#### Cocoapods
pod 'YGCVideoToolbox'

* Swift 4
